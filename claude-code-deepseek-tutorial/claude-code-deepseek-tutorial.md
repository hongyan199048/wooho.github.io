# 手把手教程：在 Trae 里用 Claude Code（图形界面），通过 ccswitch 接入 DeepSeek

> **这篇写给谁看**：Windows 电脑、**几乎没用过终端的新手**、使用 **Trae**（而非 VSCode）编辑器。
> 全程**以点鼠标为主，基本不敲命令**。每一步都会告诉你「点哪里、填什么、看到什么算成功」。
>
> **做完你能得到什么**：在 Trae 里拥有一个 AI 编程助手（Claude Code 图形聊天面板），后台用的是**便宜很多的 DeepSeek API**，而非官方按 Claude 收费的接口。
>
> **一个前提要讲清楚**：Claude Code 本身是为 VSCode 设计的插件，但 Trae 和 VSCode 底层是同一套架构，**插件安装、设置、使用方式完全一样**。实际操作中我们发现在某些机器上 **Trae 比 VSCode 更稳定**（这也是本教程用 Trae 的原因），二者按本文操作没有任何区别。
>
> 最后更新：2026-07 · 系统：Windows 10 / 11 · 编辑器：Trae

## 先花 1 分钟看懂这几个词

不用背，看一眼有印象就行：

| 名词 | 一句话解释 |
| --- | --- |
| **Trae** | 字节跳动出的免费 AI 代码编辑器，长得和 VSCode 很像，**底层也是同一套架构**，所以 VSCode 的插件和设置在 Trae 里完全通用。 |
| **Claude Code 插件** | 装进 Trae 的扩展，提供一个**图形聊天面板**（有输入框、能对比代码改动），你像聊天一样让它帮你写代码。 |
| **API Key** | 一串密码（形如 `sk-xxxx`），相当于你在 DeepSeek 的「门禁卡 + 付费凭证」。**要保密，不可分享。** |
| **ccswitch（cc-switch）** | 一个带界面的开源小工具，帮你**点几下鼠标就把 Claude Code 的后台从官方接口切到 DeepSeek**，不用手改配置文件。 |

> 💡 **这张图帮你理解整个结构**：
>
> ```
> Trae 编辑器  ←→  Claude Code 插件（图形聊天面板）  ←→  cccswitch（改了一份配置）  ←→  DeepSeek 服务器
> ```
>
> ccswitch 做的事很简单：把 Claude Code 的「后台地址」从官方换成 DeepSeek 的。Claude Code 以为自己还在连官方，但其实走了 DeepSeek，费用便宜很多。

## 整体流程

```
装 Trae  →  装 Claude Code 插件  →  打开一次面板  →  装 cc-switch
    →  申请 DeepSeek 密钥  →  在 cc-switch 里填好并启用  →  打开插件发消息验证 🎉
```

全程约 **15~25 分钟**，不需要编程基础，不需要碰命令行。

---

## 第 1 步：安装 Trae（已装可跳过）

Trae 是字节跳动推出的免费 AI 代码编辑器，原生支持中文界面，对国内用户更友好。

1. 浏览器打开 Trae 官网：<https://www.trae.ai>
2. 下载 **Windows 版**（`.exe` 安装包）。
3. 双击安装包，一路「下一步 → 我同意 → 安装」即可。
4. 安装完成，打开 Trae。界面默认是中文的。
   - 如果打开是英文的，可以在右上角设置里切换语言。

> 🖼️ **图片 1 — Trae 官网下载页面**
>
> 截图内容：Trae 官网首页，**用红框圈出「Download for Windows」按钮**（或中文「下载 Windows 版」）。
> 目的：让新手知道该点哪个按钮，避免点到其他平台版本。

## 第 2 步：在 Trae 里安装 Claude Code 插件

这一步装的是**图形聊天面板**——你后面用的就是它。Trae 底层兼容 VSCode，插件安装方式完全一致。

> 💡 **关键提示**：Trae 默认用的是 **Open VSX** 应用商店。如果搜不到 Claude Code，可尝试在设置中将应用商店切换为 VS Code 官方市场。

### 在插件商店搜索安装

1. 打开 Trae。
2. 看左边竖排的一列图标，找到**由四个小方块组成**的那个图标（「扩展」），点它。
   - 或者按快捷键 **`Ctrl + Shift + X`** 直接打开。
3. 顶部搜索框输入 **`Claude Code for VS Code`**。
4. 找到发布者是 **Anthropic** 的结果，点 **「安装」**。

> 标注要点：
> - 圈出左边竖排图标栏里的「四个小方块」扩展图标
> - 圈出顶部搜索框，里面显示 `Claude Code for VS Code`

## 第 3 步：先打开一次 Claude 面板

**这一步很重要，不要跳过。** 目的是让电脑自动生成 Claude Code 的配置文件夹，好让后面的 cc-switch 能找到它。

1. 在 Trae 里先随便打开一个文件夹：菜单「文件 → 打开文件夹」，选桌面上任意一个文件夹即可。
2. 点右上角的 **✱ 星形图标**，打开 Claude 面板。
3. 面板会弹出一个**「登录 / Sign in」**画面。
   - **不用理会登录**。因为后面走的是 DeepSeek，**接入 DeepSeek 后不需要登录也能直接用**。cc-switch 配好之后插件会自动跳过登录。
   - 看到这个画面就说明配置文件夹已经在后台自动生成了，目的达成。
   - 这个配置文件夹的位置是：`C:\Users\你的用户名\.claude\`，先关掉面板，后面用。

> 🖼️ **图片 4 — Claude 面板弹出的登录界面**
>
> 截图内容：Trae 里 Claude 面板首次打开时显示的「Sign in / 登录」画面。
> 标注要点：
> - 红框圈出「Sign in」按钮（或登录提示文字）
> - 在旁边用大号文字标注：「✅ 看到这个页面就对了！如果只用 DeepSeek，不用管这个登录。cc-switch 配好之后会自动跳过登录。」
> - 底部小字标注配置文件夹路径 `C:\Users\你的用户名\.claude\`

## 第 4 步：安装 cc-switch（切换工具）

cc-switch 是一个开源的**带图形界面**的小工具，专门给 Claude Code 切换后台供应商用。由 GitHub 上的 **farion1231** 开发维护。

1. 浏览器打开：<https://github.com/farion1231/cc-switch>
2. 在页面右侧找到 **「Releases」**（最新版本），下载 **Windows 版安装包**（文件名带 `.exe` 或 `setup`）。
3. 双击安装，一路下一步。
4. 打开 cc-switch。首次启动时它会**自动扫描你电脑上的 Claude Code 配置并导入**。
5. 右下角任务栏里会出现 cc-switch 的小图标，说明它在运行。

> 如果 cc-switch 提示「找不到 Claude Code」→ 多半是[第 3 步](#第-3-步先打开一次-claude-面板)漏了。回 Trae 打开一次面板，再重开 cc-switch。

> 🖼️ **图片 5 — cc-switch 的 GitHub Releases 下载页面**
>
> 截图内容：GitHub 仓库右侧栏，**红框圈出「Releases」链接**；以及点进去后的最新版本页面，**红框圈出 Windows 版 `.exe` 下载链接**。
> 目的：新手很容易在 GitHub 页面迷路，不知道去哪里下载。

## 第 5 步：申请 DeepSeek API Key

这是你在 DeepSeek 的「门禁卡 + 付费凭证」。

1. 浏览器打开 DeepSeek 开放平台：<https://platform.deepseek.com/api_keys>
2. 注册 / 登录（手机号或邮箱均可）。
3. **先去「充值 / 余额」充点钱**（比如 10 元，足够用很久）。**不充值的话后面调用会报错。**
4. 点 **「创建 API Key」**，随便起个名字（如 `claude-code`）。
5. 生成后会显示一串 `sk-xxxxxxxx` —— **马上复制，存到记事本里**。
   - ⚠️ **这串密钥只显示这一次，关掉页面就再也看不到了。务必先保存好。**
   - ⚠️ 不要泄露给任何人，不要发到聊天群/朋友圈/截图里。

## 第 6 步：在 cc-switch 里填 DeepSeek 并启用

回到 cc-switch 的窗口。cc-switch 已经内置了 DeepSeek 的配置模板，**你只需要填 API Key 就行**，Base URL 和模型名都会自动填好。

1. 点右上角的 **「+」**（新增 Provider）。
2. 在模板列表里选 **DeepSeek**。选完之后 Base URL、模型名都已经帮你填好了，不用动。
3. 在 **API Key** 栏里，粘贴[第 5 步](#第-5-步申请-deepseek-api-key)存的 `sk-xxxx……`（注意：整串完整，**别多带空格或换行**）。
4. 点 **「测试 (Test)」** 按钮跑个测试请求，显示通过后再点 **「启用 (Enable)」**。

> 🖼️ **图片 8 — cc-switch 添加 DeepSeek 供应商**
>
> 截图内容：cc-switch 的「新增 Provider」表单，已选择 DeepSeek 模板。
> 标注要点：
> - 黄框圈出模板列表中的「DeepSeek」选项，标注「选这个，其他自动填好」
> - 红框圈出 **API Key** 栏，指向旁边「⚠️ 粘贴你的 sk-xxxx，只填这一栏就够了」
> - 绿框圈出下方的 **「测试」** 和 **「启用」** 按钮，标注：「先点测试，通过了再点启用」
> - 其他已自动填好的字段（Base URL、模型名）用灰色淡化或标注「自动填好，不用改」

> ⚠️ **提醒**：cc-switch 内置的 DeepSeek 模板使用的是 `deepseek-v4-pro` 和 `deepseek-v4-flash`（旧模型名 `deepseek-chat` / `deepseek-reasoner` 已于 2026 年 7 月 24 日弃用，模板中不会出现）。如果你用的是旧版 cc-switch 且看到了旧名字，请升级到最新版。

> 💡 **cc-switch 实际改了哪个文件？**（了解即可，不用手动操作）
>
> 它把配置写进了 `C:\Users\你的用户名\.claude\settings.json`。这份文件正是 Claude Code 插件读取的那份，所以配一次，插件面板就能用。

## 第 7 步：验证是否接通了 DeepSeek

cc-switch 启用之后，直接在 Trae 里打开 Claude Code 插件即可使用，不需要额外操作。

1. 点右上角 **✱ 星形图标**打开 Claude 面板。
2. 直接就是聊天输入框，输入一句测试消息，例如：

   ```
   你好，请用一句话介绍你自己
   ```

3. **它正常回复了** → 🎉 成功！你现在用的就是 DeepSeek 后台，可以开始干活了。

> 🖼️ **图片 9 — 验证成功！聊天面板正常回复**
>
> 截图内容：Trae 里 Claude Code 面板，输入框里有 `你好，请用一句话介绍你自己`，下方是 AI 的正常回复内容。
> 标注要点：
> - 绿框圈出 AI 的回复，旁边标注：「🎉 能正常回复 = 已接通 DeepSeek！」
> - 底部小字：「如果没有要求登录、直接就能对话，说明一切配置正确」
